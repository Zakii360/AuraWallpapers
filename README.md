# AuraWallpapers

Open-source live wallpaper engine for Windows and Linux.

AuraWallpapers allows you to use videos, web pages, and interactive wallpapers as your desktop background.

Built with Electron and web technologies.

---

## Features

- Video wallpapers (.mp4 only so far)
- HTML/CSS/JavaScript wallpapers
- Interactive canvas wallpapers
- Wallpaper library
- Multi-monitor support (planned)
- Performance controls
- Windows and Linux support
- Open wallpaper format

---
